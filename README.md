# SUMA-RESTA-en-Primer-Grado-Un-Taller-que-Transforma
¿Para qué estamos aquí hoy? Que cada docente salga con al menos una idea que cambie lo que hace mañana en su aula: comprendiendo cómo construyen los niños de primer grado el pensamiento aditivo, con estrategias concretas para enseñar suma y resta desde el sentido numérico, y con la convicción de que el error es el camino, no el obstáculo.
